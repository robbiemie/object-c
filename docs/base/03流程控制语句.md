# 流程顺序语句

> 改变程序的流程顺序

- `if` 逻辑判断

```c++
if(表达式1) {
  // 执行语句
} else if(表达式2) {
  // 执行语句
} else {
  // 执行语句
}
```

- `switch` 跳转语句

```c++
switch(表达式) {
  case 1:
    // 执行语句
    break;
  case 2:
    // 执行语句
    break;
  default:
    // 执行语句
    break;
}

```

- `while` 循环语句

```c++
while(循环条件) {
  // 执行语句
}
```

- `for` 循环语句

```c++
for(i=0;i<10;i++) {
  if(表达式)
    // 跳过循环
    continue;
  // 执行语句 
}

```
